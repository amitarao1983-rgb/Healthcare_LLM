# Nursing Department Task Manager

Streamlit app to coordinate nurse trainings, daily nursing tasks, and report
uploads for a given calendar year.

## Features

- Training calendar with priorities (High, Medium, Low) and statuses.
- Alarm center that highlights trainings starting soon.
- Daily nursing task board with wards, shifts, and due times.
- Placeholder uploads for reports or task attachments.

## Run locally

```bash
pip install -r requirements.txt
streamlit run streamlittest_app.py
```
